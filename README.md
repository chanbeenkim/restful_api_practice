# Restful API 연동 연습

Restful API 연습 연습을 해보았다.
지난 BloC 상태 관리 연습을 해보아서 쉽게 연동할 수 있었다.

## 🖍️ 순서 정리

- HTTP 패키지 설치
- 데이터 모델 및 JSON 직렬화 생성(Quicktype으로 쉽게 제작 가능하나 직접 작성했다.)
- model - provider- service 생성
- UI에 데이터 연동 및 구현
